# P-Portafolio

Personal portfolio built with React + Vite.  
Dark, cinematic hero with light rays, a horizontal skills scroller, and subtle micro‑interactions.

## Highlights
- Shiny hero title + CTA
- Horizontal, snap‑scroll skills list with edge fades
- Clean, minimal UI with custom typography

## Tech
- React 18
- Vite
- Tailwind (base setup) + custom CSS

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
